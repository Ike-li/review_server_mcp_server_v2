# Codex Review Records

This directory stores Codex code review outputs for this repository.

## Naming

- One review per file
- File name format: `review_YYYYMMDD_HHMMSS.md`
- Use local repository time when generating the timestamp

## Required Sections

Each review file should include:

1. Review time
2. Review scope
3. Overall correctness
4. Overall explanation
5. Findings

If a review has no actionable findings, keep the file and write `No actionable findings.` in the findings section.
